E-Commerce Sales Intelligence Dashboard

Overview
This project is a Flask-based web dashboard that analyzes e-commerce sales data and provides insights using charts and reports.

---

Features
- KPI Metrics (Sales, Orders, Avg Value)
- Sales by Category
- Monthly Trends
- Customer Insights
- Top Products
- CSV Download
- PDF Report Generation

---

Tech Stack
- Python (Flask)
- Pandas, NumPy
- HTML, CSS, JavaScript
- Chart.js
- ReportLab

---

Run Project
```bash
pip install flask pandas numpy reportlab
python app.py
